## ðŸ“š Features Guide

### 1. Dashboard
**What it does**: Shows your learning progress and quick actions

**How to use**:
- View your statistics (questions asked, quizzes taken, average score)
- See recent activity
- Click quick action cards to jump to any feature
- Track your progress with trend indicators

**Tips**:
- Check dashboard daily to track progress
- Use quick actions for faster navigation
- Monitor your average score to see improvement

### 2. Ask Questions
**What it does**: Get instant AI-powered answers to any question

**How to use**:
1. Type your question in the text area
2. Choose explanation style:
   - **Simple**: Easy to understand for beginners
   - **Exam-Oriented**: Focused on exam preparation
   - **Real-World**: Practical applications and examples
3. Click "Ask Question"
4. Review the answer, topics, and concepts
5. Check the confidence score

**Example Questions**:
- "What is photosynthesis?"
- "Explain Newton's laws of motion"
- "How does machine learning work?"
- "What causes climate change?"

**Tips**:
- Be specific in your questions
- Try different explanation styles
- Review related topics for deeper understanding
- Save important answers for later review

### 3. Generate Quizzes
**What it does**: Create custom quizzes on any topic

**How to use**:
1. Enter a topic or upload a file (PDF, DOCX, TXT)
2. Select difficulty: Easy, Medium, or Hard
3. Choose number of questions (1-20)
4. Click "Generate Quiz"
5. Answer all questions
6. Submit to see your score and explanations

**Quiz Types**:
- Multiple Choice Questions (MCQ)
- True/False
- Short Answer

**Tips**:
- Start with easy quizzes to build confidence
- Upload your study materials for targeted quizzes
- Review explanations for wrong answers
- Retake quizzes to improve scores

### 4. Summarize Notes
**What it does**: Convert long texts into concise summaries

**How to use**:
1. Enter a title for your notes
2. Paste text or upload a file
3. Choose summary format:
   - **Bullet Points**: Quick key points
   - **Paragraph**: Cohesive summary
   - **Outline**: Hierarchical structure
   - **Key Concepts**: Main ideas explained
4. Click "Generate Summary"
5. Review summary and key terms

**Best For**:
- Lecture notes
- Textbook chapters
- Research papers
- Study materials

**Tips**:
- Use bullet points for quick review
- Try different formats for same content
- Highlight key terms for memorization
- Save summaries for exam preparation

### 5. Voice Assistant
**What it does**: Practice communication skills with AI

**Conversation Modes**:

#### Casual Mode
- **Purpose**: Practice everyday communication
- **Best For**: Building fluency and confidence
- **Features**: Natural conversation, encouragement
- **Example**: "Tell me about your hobbies"

#### Interview Mode
- **Purpose**: Prepare for job interviews
- **Best For**: Mock interviews and STAR method practice
- **Features**: Interview questions, constructive feedback
- **Example**: "Tell me about a time you solved a problem"

#### Presentation Mode
- **Purpose**: Improve public speaking
- **Best For**: Presentation skills and delivery
- **Features**: Feedback on clarity and structure
- **Example**: "I want to practice my presentation on climate change"

**How to use**:
1. Choose a conversation mode
2. Click "Start Session"
3. Type your message
4. Read AI response (or listen with text-to-speech)
5. Review feedback scores and suggestions
6. Continue conversation
7. End session when done

**Tips**:
- Practice regularly for best results
- Review feedback after each session
- Try all three modes
- Focus on suggestions for improvement

### 6. Profile Management
**What it does**: Customize your learning experience

**How to use**:
1. Click profile icon in navigation
2. Click "Edit Profile"
3. Update your information:
   - Full name
   - Education level
   - Grade
   - Subjects
   - Learning style
4. Click "Save"

**Why it matters**:
- Personalized learning experience
- Better AI recommendations
- Track your academic progress

## ðŸ”§ Troubleshooting

### Common Issues

**Can't log in?**
- Check email and password
- Clear browser cache
- Try different browser

**AI not responding?**
- Check internet connection
- Verify API key is set (admin only)
- Refresh the page

**Quiz not generating?**
- Check file format (PDF, DOCX, TXT only)
- Ensure file size is under 10MB
- Try with text input instead

**Voice assistant not working?**
- Check browser permissions
- Enable text-to-speech in browser
- Try typing instead of voice input
